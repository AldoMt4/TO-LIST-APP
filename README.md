    <section class="section installation">
        <h2>Installation</h2>
        <p>Follow the steps below to set up the To-Do List App locally:</p>
        <ol>
            <li>Clone the repository: <code>git clone https://github.com/AldoMt4/todo-list-app.git</code></li>
            <li>Navigate to the project folder: <code>cd todo-list-app</code></li>
            <li>If using a backend, install dependencies: <code>npm install</code></li>
            <li>Run the app: <code>npm start</code></li>
            <li>If no backend, open the HTML file directly in your browser.</li>
        </ol>
    </section>

    <section class="section usage">
        <h2>How to Use</h2>
        <ul>
            <li><strong>Add a Task:</strong> Type your task into the input box and click "Add Task" to add it to your to-do list.</li>
            <li><strong>Mark as Complete:</strong> Click the checkmark next to a task to mark it as completed. Completed tasks will be styled differently.</li>
            <li><strong>Delete a Task:</strong> Click the "Delete" button next to a task to remove it from your list.</li>
        </ul>
    </section>
